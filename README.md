# 📎 FormReady — Govt Exam Image Tool 🇮🇳

India ke sabse accurate exam image compression tool.

## Features
- ✅ UPSC, SSC, Railway, IBPS/SBI, UP Police, NEET/JEE presets
- ✅ Photo & Signature exact KB pe compress
- ✅ 11 Document types (Aadhar, PAN, Marksheet, Certificates...)
- ✅ WhatsApp sharing
- ✅ 100% browser-based — koi server nahi
- ✅ Zero blur, binary search compression
- ✅ Mobile + Tablet + Desktop responsive

---

## 🚀 Deploy Karo (Vercel — FREE)

### Step 1 — GitHub pe upload karo
1. github.com pe jaao → New Repository banao "formready"
2. Ye saare files upload karo

### Step 2 — Vercel se connect karo
1. vercel.com pe jaao → Sign up (GitHub se)
2. "Add New Project" → GitHub repo select karo
3. Settings sab default rehne do
4. **"Deploy"** click karo

### Step 3 — Live! 🎉
2 minute mein `yourname.vercel.app` pe live ho jaayega!

## 💻 Local Run Karo

```bash
# Dependencies install karo
npm install

# Development server start karo
npm run dev
# Browser mein khulega: http://localhost:5173
```

## 📦 Build Karo

```bash
npm run build
# dist/ folder ban jaayega — kisi bhi host pe upload kar sakte ho
```
